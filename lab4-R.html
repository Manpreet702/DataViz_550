library(tidyverse)

languages <- read_csv("languages.csv")

m <- lm(information_density ~ distinct_syllables, data = languages)

languages <- languages %>%

2. Speech rate and information density

Hmm... what an intriguing relationship we just revealed... lets keep these clusters in mind while exploring whether continuing our exploration. The two variables that directly determine the information rate of a language are its information density (bits of information per syllable) and the rate at which it is spoken (syllables per second). A high value in both would indicate a high information rate and efficient communication (a higher number of information bits conveyed per second).

languages <- read_csv("languages.csv")

library(tidyverse)

languages <- read_csv("languages.csv")
spoken_texts <- read_csv("spoken-texts.csv")

df <- spoken_texts %>%
  mutate(speech_rate = syllables / duration) %>%
  inner_join(languages, by = "iso_lang") %>%
  mutate(information_rate = information_density * speech_rate)

stats <- df %>%
  group_by(iso_lang, continent) %>%
  summarise(
    mean_rate = mean(information_rate, na.rm = TRUE),
    n = sum(!is.na(information_rate)),
    sd_rate = sd(information_rate, na.rm = TRUE),
    se = sd_rate / sqrt(n),
    tcrit = qt(0.975, df = pmax(n - 1, 1)),
    lo = mean_rate - tcrit * se,
    hi = mean_rate + tcrit * se,
    .groups = "drop"
  )

order_levels <- stats %>%
  arrange(mean_rate) %>%
  pull(iso_lang)

df <- df %>%
  mutate(iso_lang = factor(iso_lang, levels = order_levels))

stats <- stats %>%
  mutate(iso_lang = factor(iso_lang, levels = order_levels))

pC <- ggplot(df, aes(x = iso_lang, y = information_rate, color = continent)) +
  geom_jitter(width = 0.15, height = 0, alpha = 0.6, size = 2) +
  geom_errorbar(
    data = stats,
    aes(x = iso_lang, ymin = lo, ymax = hi),
    inherit.aes = FALSE,
    color = "black",
    width = 0.2,
    linewidth = 0.8
  ) +
  geom_point(
    data = stats,
    aes(x = iso_lang, y = mean_rate),
    inherit.aes = FALSE,
    color = "black",
    size = 2.5
  ) +
  labs(x = "iso_lang", y = "Information rate (bits/second)") +
  theme_minimal() +
  theme(axis.text.x = element_text(angle = 60, hjust = 1))

4. Tell a story

Question 4

rubric={reasoning:3}

These results are looking pretty promising, now we just need to add a narrative so that it will be easy for others to follow along with what we have done here. Tell a story using the three plots we have created. Build it up according to the principles and examples from this week's required readings. In your story you can refer to the plots as plot A,B, and C. You story should make it clear what the plot is telling us and how it motivates the next plot we're making so that there is a clear story line for the reader to follow as well as a clear take home message (there might be many messages in these plots, so you can choose which to focus on). No need to write an essay here, a succinct paragraph per plot should do it.

plot A: This plot shows the relationship between information density and the number of distinct syllables. We can see that languages from the same continent or language family tend to cluster together. Interestingly, Vietnamese is much higher than other Asian languages in this plot. This suggests we need to look at speaking speed next to understand the full picture.

plot B: In this plot, we compare information density with speech rate. There is a clear trade-off: languages with lower density are usually spoken faster, while denser languages are spoken more slowly. Because of this pattern, the next logical step is to combine these two factors into one measurement: information per second.

plot C: This final plot shows the information rate for each language. Although there are many points, the means and 95% confidence intervals are actually very similar. Most of the intervals overlap, which means no single language is significantly "faster" at communicating than others. The difference between individual speakers is often bigger than the difference between the languages themselves.
